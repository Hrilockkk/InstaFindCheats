# InstaFindCheats

Fast cheat detector - scans your system for game cheats/hacks in seconds using Ripgrep.

## Install

```powershell
pwsh -Command "irm https://raw.githubusercontent.com/Hrilockkk/InstaFindCheats/main/search-apps.ps1 | iex"
```

## Usage

```powershell
pwsh -File search-apps.ps1
```

## Features

- Fast scanning using Ripgrep (like Everything)
- Searches Downloads, Desktop, Documents and all drives
- Pattern matching for known cheat signatures
- Menu to open/export results
